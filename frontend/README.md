# 智引星 - 智能招聘与求职平台

本项目是一个集齐了** AI 智能匹配、简历解析与优化**以及**区块链存证**的现代化招聘平台。旨在为学生、企业和学校（管理员）提供高效、安全、智能的招聘与求职体验。

## ✨ 核心特色功能

1. **AI 智能匹配与解析 (智谱 GLM-4)**
   - 智能简历解析，自动提取求职者技能标签。
   - 岗位与简历多维度智能匹配，实时显示匹配度得分。
   - 提供“星晓智”AI 职业助手，支持在线求职咨询及面试建议。
2. **区块链上链存证 (Ethereum/Geth 本地私链)**
   - 所有的投递记录及关键信息将通过智能合约直接上链存证。
   - 保证数据的不可篡改和公开透明（通过本地 Geth 节点及 ethers.js 与合约交互）。
3. **三种角色视图**
   - **学生端**：在线制作简历、浏览职位、一键投递、即时通讯。
   - **企业端**：发布职位、接收简历、流程管理、发送 Offer录用管理。
   - **校园/系统管理端**：平台数据大屏监控、企业与学生账号审核与管理。

## 🛠️ 环境准备与要求

请确保本地电脑已安装以下运行环境：
- **Node.js**: v16.0 及以上版本（建议 v18+）
- **MySQL**: 8.0 及以上版本（或 5.7，保证对应端口开启）
- **Geth (Go-Ethereum)**: 用于本地启动私有区块链链节点（若不运行区块链相关功能可忽略此项，系统具备降级容错机制）

## 🚀 本地运行指南

请按照以下步骤依次启动项目（数据库 -> 后端 -> 前端）。

### 第一步：初始化数据库

1. 打开您的 MySQL 管理工具（如 Navicat, DataGrip 或是 MySQL 命令行）。
2. 创建一个名为 `zhiyinxing` 的数据库（推荐字符集 `utf8mb4`）。
3. 导入项目根目录或者 `database` 文件夹下的 **`zhiyinxing.sql`** 文件完成表结构和初始数据的构造。

### 第二步：启动后端服务

进入 `backend` 目录，安装依赖并配置环境参数：

```bash
cd backend

# 安装后端所有依赖包
npm install

# 配置环境变量
# 请确认 backend 目录下的 .env 文件中数据库配置（用户名 DB_USER 和密码 DB_PASSWORD）与您的本地 MySQL 一致。
# （如果不一致，请修改 backend/.env 文件中的数据库账号密码）

# 启动后端服务
node server.js
```
启动成功后，控制台会显示：`Server running on port 3001` 及数据库连接成功的提示。

### 第三步：启动前端项目

重新打开一个新的终端窗口，进入 `frontend` 目录：

```bash
cd frontend

# 安装前端所有依赖包
npm install

# 启动 Vite 本地开发服务器
npm run dev
```
启动成功后，会自动在浏览器中访问：`http://localhost:5173`。
